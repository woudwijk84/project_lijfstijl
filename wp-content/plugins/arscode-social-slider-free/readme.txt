== Installation ==

1. Login to WordPress admin panel, go to 'Plugins', next click 'Add New', choose 'Upload', click 'Browse' and find a arscode-social-slider-free.zip and next click 'Upload Now'

2. If plugin is installed, go to 'Settings' choose 'Social Slider' and now you can configure the plugin.

For more help go to 
[plugin homepage](http://arscode.pro/).
